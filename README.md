# Ychebnaya_Praktika123

учебный проект для учебной практики.
проект реализует библиотеку операций над плотными матрицами и демонстрационное приложение с графическим интерфейсом.

## структура проекта

- `cpp_core/`
  c++20 библиотека для работы с матрицами:
  - сложение и вычитание
  - умножение матриц
  - умножение на скаляр
  - транспонирование
  - вычисление следа
  - вычисление определителя (для малых размеров)
  - lu-разложение (решение слау)

- `ios/MatrixSwiftDemo/`
  ios-приложение на swiftui, демонстрирующее работу с матрицами.
  приложение выполняет реальные вычисления (умножение матриц) и используется
  для показа работы алгоритмов на мобильной платформе.

## используемые технологии

- c++20 (stl, оптимизированные алгоритмы)
- swift 5
- swiftui
- xcode
- macos / ios sdk

## назначение проекта

проект разработан в учебных целях и демонстрирует:
- разработку вычислительных алгоритмов
- модульную архитектуру
- перенос алгоритмов на мобильную платформу
- работу с графическим интерфейсом
