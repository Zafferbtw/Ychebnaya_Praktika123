# MatrixSwiftDemo

ios-приложение для демонстрации операций над матрицами.
реализовано на swiftui.

приложение позволяет:
- задать размер квадратной матрицы n×n
- ввести элементы двух матриц
- выполнить умножение матриц
- получить численный результат вычислений

все вычисления выполняются реально, без визуальных заглушек.

## требования

- macos
- установленный xcode (через app store)
- ios simulator (устанавливается вместе с xcode)

## запуск приложения

1. открыть xcode
2. создать новый проект:
   - app
   - interface: swiftui
   - language: swift
   - name: MatrixSwiftDemo
3. заменить файл `ContentView.swift` на версию из этого проекта
4. выбрать любой ios simulator (например, iPhone 14)
5. нажать **Run (▶)**

## функциональность

- ввод матриц в текстовом виде (построчно)
- изменение размерности матрицы через stepper
- вычисление произведения матриц
- вывод результата в удобочитаемом виде

## ограничения

- поддерживаются квадратные матрицы
- корректность ввода данных предполагается на стороне пользователя
- максимальный размер n ограничен значением, заданным в интерфейсе (по умолчанию до 10)
